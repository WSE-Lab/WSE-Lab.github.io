MoRE

MoRE Demo

MoRE is a CARLA-based tool for defining and evaluating traffic rules for autonomous driving systems. This demo shows how to create a local rule suite with a parameterized speeding rule and a solid-yellow-line lane-change rule. It then demonstrates offline evaluation, parameter adjustment, and live evaluation during a CARLA simulation. MoRE links detected violations to the corresponding simulation frames and exports the complete results as a self-contained HTML report.
